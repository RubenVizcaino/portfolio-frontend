Ecommerce Fullstack Template

Proyecto base e-commerce con arquitectura limpia: Angular en el frontend, Spring Boot + JPA + MariaDB en el backend, y Lombok para simplificar el modelo de datos. Basado en patrón Controller–Service–Repository para mantener la separación de responsabilidades. Contectado a una API REST para operaciones CRUD.
